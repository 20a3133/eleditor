# テキストエディタ　デモアプリ
研究の一環でElectronを使ってテキストエディタを作りました  
内容はコードのみです  
githubの転送容量の都合上Electronのモジュールをpushできなかったので実行前に  
**npm install --save-dev electron**  
を実行してElectronのインストールをお願いします
